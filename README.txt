Hexagonal Sandbox - песочница с процедурной генерацией ландшафта, мир которой полностью состоит из шестиугольных тайлов.
Игрок может свободно перемещаться по миру игры и изменять его, ломая и ставя тайлы.
Все изменения сохраняются (при корректной работе программы).

Мир игры хранится в виде чанков, квадратных областей 16х16 тайлов. Чанки создаются на ходу, поэтому гипотетически мир игры бесконечен.
Ландшафт мира генерируется хаотично, используя функции шумов и ГПСЧ. За основу всех вычислений берётся сид, случайно генерирующийся при создании
мира. Два мира с одинаковым сидом будут обладать абсолютно идентичной натуральной генерацией.
Игра использует функции шумов Перлина, одномерный шум Перлина для разнообразия высоты поверхности и распределения биомов и двумерный шум Перлина для генерации пещер.
Чем глубже спускается игрок, тем просторнее становятся генерируемые пещеры.

Каждый чанк хранит 256 тайлов. 
Тип тайла задаётся тремя параметрами: основным материалом, покрывающим материалом (если есть) и формой.

Материал тайла определяет его цвет, звук и яркость, если он является источником света.
Список всех существующих в игре материалов:
-Воздух.
-Трава. Генерируется покровом в биомах равнин. Также является первым по умолчанию доступным игроку для строительства материалом.
-Лишайник. Может сгенерироваться в каменных и гранитных равнинах если там окажется какая-либо плодородная почва.
-Земля. Является плодородной почвой. Генерируется в равнинах.
-Камень. Генерируется везде ниже слоя почвы, а так же в каменных равнинах.
-Снег. Генерируется покровом в тундре.
-Замёрзшая земля. Является плодородной почвой. Генерируется в тундре.
-Песок. Является плодородной почвой. Генерируется в пустыне.
-Гранит. Генерируется в гранитных равнинах.
-Перегной. Является плодородной почвой. Генерируется на светящихся землях.
-Светящаяся трава. Генерируется на светящихся землях. Излучает тусклый свет.

Форма тайла определяет основу его текстуры.
Список всех возможных форм:
-Пустота
-Полный блок
-Полублок (нижняя половина - основной материал, верхняя - покрывающий)
-Полублок (верхняя половина - основной материал, нижняя - покрывающий)
-Стена
-Тайл с покровом
Игрок может менять форму тайла, нажимая по нему правой кнопкой мыши.

Покрывающий материал определяет второй материал тайла для таких форм как тайл с покровом и полублок.
Ломая тайл, игроку становятся доступны для строительства материалы, из которых этот тайл состоял.

На поверхности выбор материалов, использующихся в генерации ландшафта, зависит от природной зоны (биома) чанка.
Также биом влияет на цвет неба.
Список всех существующих в игре биомов:
-Равнины. Самый обычный биом, в качестве почвы генерируется земля, а в качестве покрова - трава
-Тундра. Почвой является замёрзшая земля, покрывается снегом.
-Пустыня. Вместо почвы песок, покров отсутствует.
-Каменные равнины. Вместо почвы камень, в нечастых случаях, когда в чанке с этим биомом генерируется земля, она покрывается редким растением - лишайником.
-Гранитные равнины. То же, что и каменные, но вместо камня гранит.
-Светящиеся земли. Самый редкий биом. Почвой здесь является перегной, на котором растёт светящаяся трава. Небо здесь имеет тёмно-синий окрас.

Миры хранятся на диске компьютера по следующей структуре:
-saves/
|-<Название мира>/
|--map/
|---[файлы чанков 'x;y.chunk']
|--meta.txt
|---[сид мира, координаты камеры, доступные игроку материалы]


УПРАВЛЕНИЕ:

МЕНЮ ВЫБОРА МИРА
| Используйте колёсико мыши чтобы прокрутить список миров
| Интерфейс:
| | Слева - список созданных миров с кнопками удаления миров
| | | Чтобы зайти в мир, нажмите на него в списке
| | | Чтобы удалить мир, нажмите на кнопку с крестиком рядом с миром в списке
| | Справа снизу - кнопка "New world"
| | | Нажмите чтобы создать новый мир и зайти в него

ИГРА
| Нажмите левую кнопку мыши чтобы сломать тайл, на который наведён курсор, или поставить тайл из выбранного материала в место курсора, если курсор наведён на пустой тайл
| Нажмите правую кнопку мыши чтобы изменить форму тайла, на который наведён курсор
| Используйте колёсико мыши чтобы изменить выбранный материал для строительства
| Используйте WASD для перемещения по миру
| Функциональные клавиши:
| | Нажимайте F3 чтобы включать и выключать экран отладки
| | Нажимайте F8 чтобы включать и выключать музыку
| | Нажимайте F11 чтобы включать и выключать режим полного экрана (работает не со всеми драйверами)
